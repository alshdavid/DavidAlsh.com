Rides of a Lifetime

A Holiday Recreation Provider

Description:



This site was produced for a company that provides recreational activities for tourists. They needed a website where they could advertise their tours, events and photo gallery.

The gallery features a masonry grid image style, with a full screen image viewer.



Features:


  • Responsive
  • Homepage slider
  • Masonry Gallery
  • CSS Animationn
  • JavaScript Animations.


Technologies:
