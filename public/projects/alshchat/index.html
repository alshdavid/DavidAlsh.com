alshchat.xyz

Description:



Alshchat was an application designed to showcase my skills with node.js, websockets, solving problems along with demonstrating my ability wrangling front end.

The focus here is the back-end as it was using a non-relational database, and being node.js, using new ES6 syntax and features - adhering to MVC principles and structuring the application in a way that makes it scalable.

I used a little functional programming here, but mostly just promises. In the “refactoring” branch, I use generator functions in combination with the co library to help manage asynchronous functions in a more readable way.

Of course now I use async/await - transpiling from either Babel or TypeScript.
