Belton I.T.

Auckland Based I.T. Support Company

belton.ongoing.co.nz

Description:



Belton IT is a great showcase of my ability to put together websites that incorporate both HTML elements and lots of JavaScript under pressure. I was given 16 hours to complete the design given to me by the designer.

The design was a responsive, handling mobile, tablet, laptop and desktop. It had many dynamic elements interacting with JavaScript also using systems like flexbox.



Features:


  • Responsive
  • Homepage slider.
  • Link that fades into a cinema page for the video.
  • Sticky nav-bar.
  • Nav-bar status changes depending on where you are on the screen.
  • A “services” selector.
  • CSS animations.
  • JavaScript Animations.


Technologies:


  • HTML5
  • SCSS
  • JavaScript
  • JQuery
  • Laravel
